# MVP Scope

## Included

- Custom authentication and protected app shell
- Client CRUD with uploads, cadres, effectifs, color and responsible consultants
- Consultant CRUD
- Calendar with mission creation/edit/delete
- Mission types: audit, formation, assistance
- Mission modes: en ligne, présentielle
- Automatic ordre de mission generation for new missions
- Ordres de mission list, edit, export Excel, PDF and print
- Reusable ordre de mission templates
- Supabase Storage for documents

## Deferred

- OCR extraction from organigramme images/PDFs
- Advanced dashboards
- Push notifications
- Google Calendar synchronization
- Email sending
- Full offline synchronization
- Invoices/facturation
