## UI use

- Primary blue is used for primary actions, selected navigation, focus rings, and small accents.
- Gray is used for secondary UI, dividers, muted text and borders.
- Large backgrounds should stay light and calm.
- Avoid unnecessary gradients.
- Shadows should be subtle.
